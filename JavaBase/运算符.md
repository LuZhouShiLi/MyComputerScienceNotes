## 二、+ 操作

* 字符的+操作 字符对应数值进行转换 然后再进行运算
```java
package com.ustc.base.DataFormat;

public class test4 {
    public static void main(String[] args) {
        int a = 1;
        char b = 'a';
        System.out.println(a + b);
    }
}

```

* 字符串+操作
字符串的+操作表示连接符
